# AI Builder Routing Rule

When the user requests practical office automation, Windows scripting, lightweight browser tools, data workflows, or security-aware code generation, use the `SKILL.md` routing hub at the repository root.

Before writing code:

1. Read `SKILL.md` for the routing contract.
2. Read the matching file under `references/builders/` (VBA, PowerShell, Python, HTML/CSS/JS, office-script, security).
3. Read `references/rules/security-baseline.md` when the task touches secrets, external content, file deletion, bulk rename, COM automation, email sends, or command execution.
4. Read `references/rules/risk-trigger-matrix.md` when the risk profile is implicit or mixed.
5. Read `references/platforms/antigravity.md` if Antigravity-specific behavior matters.

Deliver the response using the contract in `references/rules/output-and-safety.md`: brief plan, complete executable code, explicit run instructions, validation method, rollback guidance for destructive changes.

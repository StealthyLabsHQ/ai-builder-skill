# v0.6.0 - Full language spectrum (25 builders)

This release completes the language coverage for AI Builder with sixteen new builders spanning the full practical programming language spectrum.

## Highlights

### New builders (16)

| Builder | Coverage |
|---|---|
| `c-cpp-builder.md` | C and C++ systems, embedded, CLI |
| `csharp-builder.md` | .NET 8, Windows Forms, ASP.NET Core, VSTO, Unity |
| `java-builder.md` | Spring Boot, Android, enterprise JVM |
| `php-builder.md` | Laravel, WordPress, server-side scripts |
| `sql-builder.md` | Standalone SQL migrations, reporting queries, schema management |
| `ruby-builder.md` | Automation scripts, Rails, Sinatra, Rake |
| `vb-net-builder.md` | Windows Forms, legacy VB.NET maintenance |
| `r-builder.md` | Statistical analysis, ggplot2, Quarto reports |

| `go-builder.md` | CLI tools, REST APIs, single-binary cross-platform tools |

| `lua-builder.md` | Roblox/Luau, Love2D, Neovim plugins, embedded Lua |

### New pattern libraries (15)

One pattern file per builder (excluding Objective-C, which redirects to Swift for new code):
`c-cpp-patterns.md`, `csharp-patterns.md`, `java-patterns.md`, `php-patterns.md`, `sql-patterns.md`, `ruby-patterns.md`, `vb-net-patterns.md`, `r-patterns.md`, `swift-patterns.md`, `rust-patterns.md`, `go-patterns.md`, `kotlin-patterns.md`, `elixir-patterns.md`, `scala-patterns.md`, `lua-patterns.md`

### New recipes (5)

- `csharp-excel-report.md` — .NET 8 console app: scans CSVs, generates formatted Excel with EPPlus
- `r-csv-report.md` — R analysis script + Quarto HTML report with ggplot2 bar chart and summary table
- `go-cli-tool.md` — Go CSV folder aggregator with cobra, outputs JSON summary by region
- `ruby-file-organizer.md` — Ruby file organizer by last-modified date with dry-run and rollback CSV
- `sql-migration.md` — Versioned UP/DOWN migration with schema_migrations tracking and shell apply script

## Why this matters

AI Builder now covers 25 programming languages across 6 AI runtimes with consistent depth: builder (routing) → patterns (idioms) → recipes (full implementations). Agents produce idiomatic, production-safe code across the entire practical language spectrum on the first pass.

## Installation

### Claude Web Skills (recommended)

1. Download **`ai-builder.zip`** from the release assets below.
2. Go to [claude.ai](https://claude.ai) → Settings → Skills → Upload Skill.
3. Upload the zip — no further configuration needed.
4. Invoke with `/ai-builder <your request>` in any Claude.ai conversation.

### Claude Code (slash command)

```bash
# After cloning this repository into your Claude Code skills path:
/ai-builder build a React KPI dashboard with CSV drag-and-drop
```

### ChatGPT Custom GPT

Paste `dist/chatgpt-custom-gpt.md` as the system prompt in a new Custom GPT configuration.

### Codex CLI

```bash
$ai-builder write a Go CLI to aggregate CSVs by region
```

### Gemini CLI

The `GEMINI.md` root adapter imports `AGENTS.md` automatically when this repository is in context.

### Antigravity

The `.antigravity/` directory is read automatically when this repository is the active project.

## Upgrade notes

For Claude Web Skills users: download the new `ai-builder.zip` from this release and re-upload to replace the previous version. The zip contains all 25 builders, 23 pattern libraries, and 16 recipes.

# v0.1.0 - Initial public release

AI Builder is now available as a public routing-first skill repository for practical business automation.

This first release focuses on making common internal automation requests easier to route, safer to implement, and easier to hand off.

## Highlights

- Root orchestration model built around `SKILL.md` and `AGENTS.md`
- Builder references for HTML/CSS/JavaScript, VBA, PowerShell, Python, and security hardening
- Shared safety and risk guidance for secrets, destructive actions, external content, and agent-driven workflows
- Public-facing repository assets including `README.md`, MIT license, banner art, and social preview asset
- Executable examples and reusable starter kits for common automation paths
- Routing fixtures, validation scripts, and GitHub Actions CI
- OpenAI-facing metadata in `agents/openai.yaml`

## Good fit

AI Builder is designed for requests such as:

- Excel cleanup and workbook automation
- Windows file and folder tasks
- lightweight internal browser tools
- CSV and export reporting workflows
- practical automation hardening and safer script generation

## Notes

This release establishes the base repository structure and the first public documentation set.
